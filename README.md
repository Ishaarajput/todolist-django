# Django Todo List Application

This is a Todo List web application built using Django.

## Features

* User registration
* User login
* Dashboard to manage tasks
* Add and view tasks
* Search tasks

## Technologies Used

* Python
* Django
* HTML
* CSS

## How to Run

1. Clone the repository
2. Run migrations
3. Start the Django server

```
git clone https://github.com/Ishaarajput/todolist-django.git
cd todolist-django
python manage.py runserver
```
